Savigation
==================

shortcut for android app like path menu,adding screen shot fuction(rooted)

Note: 安卓上的快速启动按钮，一面是手机上的应用，另一面是系统开关快捷方式。长按，双击，拖动有相应功能。 添加了截图功能，但必须root权限。pad截图会有问题。
